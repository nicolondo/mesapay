# MESAPAY · Brand Kit

Todo lo necesario para cargar la marca en **Canva** (o cualquier otra herramienta).

## Contenido

```
brand-kit/
├─ MESAPAY-Brand-Kit.pdf        ← guía visual (logo, colores, tipografías, uso)
├─ colores.txt                  ← lista de HEX lista para pegar
├─ logos/
│  ├─ MESAPAY-logo.svg          ← logo principal (vectorial, fondo claro)
│  ├─ MESAPAY-logo.png          ← logo principal (PNG transparente, 1600px)
│  ├─ MESAPAY-logo-fondo-oscuro.svg / .png   ← reverso (para fondos oscuros)
│  ├─ MESAPAY-icono-512.png / -192.png        ← isotipo (avatar, favicon)
│  └─ MESAPAY-logo.ai           ← fuente editable (Illustrator)
└─ LEEME.md                     ← este archivo
```

## Cómo cargarlo en Canva

En Canva: **Marca → Kit de marca** (Brand Kit). Para subir fuentes y guardar
varios logos necesitás **Canva Pro**.

1. **Logos** → subí los de `logos/`: el PNG principal, el de fondo oscuro y el
   isotipo. Si tu plan acepta SVG, subí también los `.svg` (escalan sin perder
   calidad).
2. **Colores de marca** → agregá los HEX de `colores.txt`. Empezá por los 6
   principales: `#C9532E` `#A8401F` `#1A1613` `#F5F1EA` `#FBF8F3` `#6B6259`.
3. **Fuentes de la marca**:
   - **Instrument Serif** (titulares) — ya está en la librería de Canva.
   - **Geist** (texto/interfaz) y **Geist Mono** (etiquetas/montos) — descargá
     los `.ttf` de Google Fonts y subilos.

## Tipografías (gratis, licencia abierta)

- Instrument Serif — https://fonts.google.com/specimen/Instrument+Serif
- Geist — https://fonts.google.com/specimen/Geist
- Geist Mono — https://fonts.google.com/specimen/Geist+Mono

En cada página: **Get font → Download all** → `.zip` con los `.ttf`.

## Colores clave

| Rol | Nombre | HEX |
|---|---|---|
| Primario | Terracota | `#C9532E` |
| Texto | Tinta | `#1A1613` |
| Fondo | Hueso | `#F5F1EA` |
| Superficie | Papel | `#FBF8F3` |
| Secundario | Apagado | `#6B6259` |

> El arte del logo usa una terracota un punto más cálida (`#C55B3B`) y tinta
> `#161311`. Para piezas pegadas al logo podés usar esos; el estándar del
> sistema es `#C9532E` / `#1A1613`.
